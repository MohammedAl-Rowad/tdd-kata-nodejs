## To run this locally

```bash
$ git clone git@github.com:MohammedAl-Rowad/tdd-kata-nodejs.git
$ cd tdd-kata-nodejs
$ npm i
```

## To run the tests

```bash
$ npm run test
```

## To run the server

```bash
$ npm run dev
```
